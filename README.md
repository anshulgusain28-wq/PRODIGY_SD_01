# Temperature Converter

## Description

Temperature Converter is a simple software development project that converts temperatures between Celsius, Fahrenheit, and Kelvin.

## Features

- Celsius to Fahrenheit and Kelvin conversion
- Fahrenheit to Celsius and Kelvin conversion
- Kelvin to Celsius and Fahrenheit conversion
- Input validation
- Simple user-friendly interface

## Technologies Used

- HTML
- CSS
- JavaScript

## How to Run

1. Open index.html in a web browser.
2. Enter a temperature value.
3. Select the unit.
4. Click Convert.
