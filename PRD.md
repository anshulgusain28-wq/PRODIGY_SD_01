# Product Requirements Document (PRD)

## Project Title
Temperature Converter Application

## Product Overview
The Temperature Converter converts values between Celsius, Fahrenheit, and Kelvin.

## Problem Statement
Users need a quick and accurate way to convert temperatures between scales.

## Goals
- Accurate conversions
- Easy user experience
- Fast response time

## Scope
### In Scope
- Celsius, Fahrenheit, Kelvin conversions
- Input validation
- Result display

### Out of Scope
- Authentication
- Database
- Cloud sync

## User Stories
- Enter a temperature value.
- Select a source unit.
- View converted results.
- Receive validation for invalid input.

## Functional Requirements
- Temperature input
- Unit selection
- Conversion processing
- Result display
- Reset functionality

## Non-Functional Requirements
- Fast performance
- Reliable calculations
- Browser compatibility

## Acceptance Criteria
1. User enters value.
2. User selects unit.
3. User clicks Convert.
4. System displays accurate results.
